# DecentRandom v0.2.0

P2P기반 탈중앙화 오픈 소스 난수 발생기

- [공식 웹사이트](https://decentrandom.com)
- Cosmos SDK v0.39.2
- Tendermint v0.33.9
- Go 1.13 이상
- Ledger Nano 시리즈 지원

## Build

- make build

## Warnings

- master branch는 업데이트를 위해 개발 branch로 구동에 사용할 수 없습니다. 최신 릴리즈를 사용하시기 바랍니다.
